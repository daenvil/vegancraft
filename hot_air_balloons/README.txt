# Hot Air Balloons (v1.0, for Minecraft 1.21.9+)

This datapack adds hot air balloons, an alternative to riding ghasts.

- This datapack was made by Daenvil, its official pages are:

  - On PlanetMinecraft: https://www.planetminecraft.com/data-pack/hot-air-balloons/
  - On my website: https://daenvil.github.io/MCDatapacks/balloons.html

- This version was published on the 6th of October 2025. Check the webpages above for possible updates.

## Installation

- Drop the datapack zip file inside the "datapacks" folder of your world folder (<your minecraft folder>/saves/<your world>/datapacks/).

## Features

Craft a balloon with leather, wool, string, blaze powder, and a ghast tear. Right click with the balloon on the ground to spawn it. You can use the balloon as you would a happy ghast: ride it using right click, attach it using a leash, etc. You can see more info on this page: https://daenvil.github.io/MCDatapacks/balloons.html
